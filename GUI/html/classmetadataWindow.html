void metadataWindow::connectSignalClicked ( void  )
protected

To connect the clicked signal to events handler.

This method connects each widgets clicked with its own signal. If you add a new clicked event, you must append its signal connection here.

Currently, the following clicked signals have been connected:

  • importXML When connected, this button sends a signal to import an XML metatada file into the metadata first viewer
  • exportXML When connected, this button sends a signal to export an XML metatada from metadata viewer into an XML file.
  • modifyXML When connected, this button sends a signal to open a secondary metadata viewer where the user will modify the metadata of the first viewer.
  • saveXML When connected, this button sends a signal to save the XML metadata into the currently loaded MXF file.
  • XMLconformance When connected, this button sends a signal to run an EBUCore conformance analysis on the metadata.

    [in]void: No params required

metadataWindow::refGladeButton ( const Glib::RefPtr< Gtk::Builder > &  refGlade)
protected

To link the glade references to the buttons.

This method loads all required glade's references to connect the metadata viewer buttons. If you add a new button to the metadata viewer into the Glade design, you must append his reference in this function.

Currently, the following buttons have been referenced :

  • importXML When clicked, this button should import an XML metatada file into the metadata viewer
  • exportXML When clicked, this button should export an XML metatada from metadata viewer into an XML file.
  • modifyXML When clicked, this button should open a secondary metadata viewer where the user will modify the metadata of the first viewer.
  • saveXML When clicked, this button will save the XML metadata into the MXF file.
  • XMLconformance When clicked, this button starts an EBUCore conformance analysis on the metadata.

    Parameters:
    [in]refGlade: This is the reference to your glade player features - It's mandatory.
    Returns:
    nothing if all is right or gtkmm-critical error/segmentation fault
    Note:
    This function and his documentation must be completed
